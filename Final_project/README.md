# Face Detection Attendance System

A Streamlit-based Face Detection Attendance System using OpenCV and LBPH Face Recognizer.

## Features

- Face Registration
- Live Face Detection
- Attendance Marking
- Attendance Report Viewing
- CSV Export
- Duplicate Prevention

## Technologies Used

- Python
- Streamlit
- OpenCV
- Pandas
- NumPy

## How to Run

```bash
streamlit run app.py
```

## Project Structure

- app.py → Main application
- known_faces/ → Registered student images
- screenshots/ → Streamlit screenshots
- attendance.csv → Attendance records
